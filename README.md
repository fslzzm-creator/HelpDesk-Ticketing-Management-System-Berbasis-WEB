# HelpDesk-Ticketing-Management-System-Berbasis-WEB

## Stack
- HTML
- Bootstrap
- MySql
- PHP
- CSS

Deskripsi
Sistem ticketing adalah alat manajemen yang mengubah permintaan atau keluhan pelanggan 
dari berbagai saluran (email, chat, telepon) menjadi tiket terstruktur dengan nomor unik. 
Fungsinya meliputi otomatisasi alur kerja, dokumentasi terpusat, pelacakan status, dan efisiensi agen untuk mempercepat respons layanan
